Components 1 & 2
Automation & Data Collection
⚙️
GitHub Actions + Scheduler

How it starts automatically every morning

📄 daily-digest.yml
⏰Cron schedule in code. daily-digest.yml defines cron: "0 14 * * *" — which means "run at 14:00 UTC every day" = 7 AM Pacific. GitHub reads this file and fires automatically.
🖥️Free cloud VM. GitHub spins up a Linux machine, runs pip install -r requirements.txt, injects your API keys from GitHub Secrets as environment variables, then executes python digest.py. Shuts down when done.
🔑Secrets stay secure. API keys and passwords live in GitHub Secrets (encrypted). They never appear in any code file — only injected at runtime.
📡
RSS Feed Reader

How the news gets collected

📄 digest.py → fetch_articles()📦 feedparser
📰What is RSS? Every major news site publishes a machine-readable list of their latest headlines called an RSS feed. The feedparser library visits all 15 feeds and grabs the 5 newest articles from each — no login, no scraping.
📊Output: ~75 articles across 5 categories — Finance, Geopolitics, Tech, Human Insights, and AI — all formatted and passed as one big prompt to Gemini.
💰 Finance ×6 🌍 Geopolitics ×3 ⚡ Tech ×4 🧠 Human Insights ×4 🤖 AI ×4
Components 3, 4 & 5
AI Processing & Email Delivery
✦
Gemini 2.5 Flash API

The AI that reads & writes the digest

📄 digest.py → call_gemini()📦 google-genai
✦One API call, everything in. All ~75 headlines are bundled into one prompt. The prompt tells Gemini to act like a sharp, well-read friend — not a news robot. It reads everything and writes a 6-section digest with a bold headline per story.
🔁Auto-retry logic. If Gemini returns a 429 rate-limit error, call_gemini() waits 30 seconds and retries — up to 2 times before exiting cleanly.
📋6 output sections: 💰 Money Talk · 🌍 World Lore · ⚡ Tech Tea · 🧠 Human Insights · 🤖 AI · 🔥 Speed Round.
🎨
HTML Email Builder

Turns plain text into a polished email

📄 digest.py → build_html_email()
🖥️Parses Gemini's markdown, applies a dark background, color-coded section cards, bold headlines, and clean typography. Looks like a hand-designed newsletter — all in Python, no design tools.
📬
Gmail SMTP

Sends the email directly to your inbox

📄 digest.py → send_email()📦 smtplib
🔐Connects to Gmail's mail server over SSL (port 465) using a 16-character App Password — a separate credential stored in GitHub Secrets, not your real Gmail password. No third-party email service needed.
